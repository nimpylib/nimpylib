## Feature additions

### def in class
#### full support about params
e.g.
```Python
# can compile as Nim via pylib
class C:
  def f(self, a: int): return a+1
  def g(self, a) -> int: return a+1
```
#### inheritance and `super`
e.g.
```Python
class A:
  def f(self): return "A"
class B(A):
  def f(self): return super().f()
```

```Nim
template pass* = discard
template pass*(_) = discard

```
